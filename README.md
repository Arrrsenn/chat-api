# Проект "Чат"

## Описание

"Чат" — это небольшой проект, представляющий собой систему обмена сообщениями в реальном времени. Цель проекта — предоставить простой интерфейс для общения пользователей в режиме онлайн.

## Функциональность

- **Регистрация и авторизация пользователей**: Позволяет пользователям создавать аккаунты и входить в систему.
- **Отправка и получение сообщений**: Пользователи могут обмениваться текстовыми сообщениями.
- **Уведомления в реальном времени**: Сообщения отображаются в интерфейсе сразу после отправки.
- **Список контактов**: Поддержка добавления и удаления контактов.
